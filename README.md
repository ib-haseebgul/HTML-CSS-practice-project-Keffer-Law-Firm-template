## Project Overview

The **Keefer Law Firm** website is a front-end practice project that simulates a professional legal services website.  
It includes a header, navigation bar, hero section with a contact form, informational content, sidebars, and footer-style elements.

This project was built to strengthen skills in:
- Semantic HTML
- CSS layout techniques
- Page structuring
- Visual hierarchy
- Form design
- Multi-column layouts
